### What I learned

Use ARIA attributes for accessibility.

Still, some of Figma files are buggy:

- the quote paths are not well-defined
- some colors are missing - like color-card-image-border
- shadows must be guessed, or you can use a Figma plug-in like "Figma drop and inner shadow to CSS box shadow" to find
  them
